## 🎯 Quizzy

👉 https://quizz-pi-plum.vercel.app/

Quizzy est une application de quiz interactive permettant de tester ses connaissances de manière ludique et dynamique.

## ✨ Fonctionnalités

- Questions interactives
- Validation des réponses en temps réel
- Score final
- Interface simple et intuitive

## 🛠️ Technologies utilisées

- JavaScript
- HTML / CSS

## 🎯 Objectif

Créer une application interactive centrée sur l’expérience utilisateur, tout en mettant en pratique la logique de programmation et la gestion d’état.
